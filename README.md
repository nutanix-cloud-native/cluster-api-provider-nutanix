# Nutanix Kubernetes Cluster API Provider

Kubernetes-native declarative infrastructure for Nutanix.

## What is the Nutanix Cluster API Provider?

The [Cluster API](https://github.com/kubernetes-sigs/cluster-api) brings declarative Kubernetes-style APIs to cluster creation, configuration and management. The API itself is shared across multiple cloud providers allowing for true hybrid deployments of Kubernetes.
